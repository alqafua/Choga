# ربط إنستغرام (خطوات الإكمال لاحقاً — يفضّل على كمبيوتر)

الكود جاهز بالكامل. ينقص فقط إعدادان في لوحة مطوّري ميتا + متغيّرين في Railway.
**سوِّ هذا على كمبيوتر (لابتوب)** — لوحة ميتا على الجوال فيها مشكلة تُضيف أقواس `<>` للرابط وتفشل.

## المعلومات الجاهزة
- **Instagram App ID:** `1529368035597025`
- **رابط الموقع:** https://choga-production.up.railway.app
- **رابط إعادة التوجيه المطلوب (Redirect URI):**
  ```
  https://choga-production.up.railway.app/api/meta/callback
  ```

## الخطوات

### 1) في لوحة مطوّري ميتا (developers.facebook.com)
1. افتح التطبيق ← منتج **Instagram** ← **إعدادات (Settings)**.
2. القسم 4: **"إعداد تسجيل الدخول إلى حساب Instagram للأعمال"** (Business Login for Instagram).
3. في خانة **Valid OAuth Redirect URIs** الصق بالضبط (بدون أي قوس `<` أو `>`):
   ```
   https://choga-production.up.railway.app/api/meta/callback
   ```
4. اضغط **حفظ / Save** (وإن وُجد زر "Save Changes" أسفل الصفحة كاملة، اضغطه أيضاً).
5. من أعلى نفس الصفحة، انسخ قيمة **Instagram App Secret** (زر "إظهار / Show").

### 2) في Railway (railway.app ← المشروع ← Variables)
أضف/تأكد من المتغيّرين:
```
INSTAGRAM_APP_ID = 1529368035597025
INSTAGRAM_APP_SECRET = «القيمة من الخطوة 1.5»
```
ثم انتظر إعادة النشر (دقيقتين).

### 3) جرّب
افتح الموقع ← دخول (Choga / 7007) ← **الإعدادات ← الحسابات** ← زر **"ربط بإنستغرام"**.
المفروض يوديك لإنستغرام ويطلب الموافقة، ثم يرجعك ويظهر **"متصل ✓"** مع اسم الحساب.

## ملاحظة أمان
- لا تكتب `INSTAGRAM_APP_SECRET` في أي مكان عام أو محادثة — فقط داخل Railway مباشرة.
- لا حاجة أبداً لكلمة سر حساب إنستغرام نفسه — الربط يتم عبر OAuth الرسمي فقط.
