# Choga — مركز المحتوى

تطبيق ويب لإدارة إعدادات وحسابات وعملاء وطلبيات متجر Choga، مبني من تطبيق React (Vite) + خادم Express.
كل البيانات (الإعدادات، العملاء، الطلبيات) تُخزَّن على الخادم في ملفات JSON، وتسجيل الدخول محمي باسم مستخدم وكلمة مرور.

## التبويبات الجاهزة

- **الإعدادات** — هوية المشروع، الحسابات على المنصات، مفاتيح API
- **العملاء** — قاعدة بيانات العملاء (الاسم، الهاتف، العنوان، ملاحظات) مع بحث وإضافة/تعديل/حذف
- **الطلبيات** — رقم الطلبية، ربطها بعميل، تفاصيل الطلب، السعر، التواريخ، والحالة (جديدة / قيد التجهيز / جاهزة للتسليم / تم التسليم / ملغية) مع فلترة وتغيير سريع للحالة

## البنية

```
client/   تطبيق الواجهة (React + Vite)
server/   خادم Express — تسجيل الدخول + حفظ/قراءة الإعدادات + تقديم الواجهة المبنية
```

## التشغيل محلياً

```bash
npm install

# تشغيل الواجهة (مع وكيل /api على الخادم)
npm run dev:client

# في طرفية أخرى — تشغيل الخادم
npm run dev:server
```

افتح http://localhost:5173 وسجّل الدخول بـ:

- اسم المستخدم: `Choga` (أو القيمة في `ADMIN_USER`)
- كلمة المرور: `7007` (أو القيمة في `ADMIN_PASS`)

## البناء والتشغيل للإنتاج

```bash
npm install
npm run build   # يبني الواجهة إلى client/dist
npm start       # يشغّل خادم Express الذي يقدّم الواجهة + الـ API
```

## النشر على Railway

1. أنشئ مشروعاً جديداً في Railway واربطه بهذا المستودع. سيتم اكتشاف `railway.json` تلقائياً
   (يُنفّذ `npm install && npm run build` ثم `npm start`).
2. أضف متغيرات البيئة التالية من تبويب **Variables**:

   | المتغير      | القيمة المقترحة                  |
   |---------------|-----------------------------------|
   | `ADMIN_USER`  | `Choga`                            |
   | `ADMIN_PASS`  | `7007`                             |
   | `JWT_SECRET`  | نص عشوائي طويل وسرّي               |
   | `DATA_DIR`    | `/data`                             |

3. لضمان بقاء البيانات بعد كل عملية نشر جديدة (Redeploy)، أضف **Volume** من تبويب
   **Settings → Volumes** واربطه بالمسار `/data` (نفس قيمة `DATA_DIR`).
4. بعد النشر، افتح رابط الخدمة وسجّل الدخول بالبيانات أعلاه. أي تغييرات تُحفظ من شاشة
   الإعدادات ستبقى محفوظة في الـ Volume بين عمليات النشر.

## ملاحظة أمنية

غيّر `ADMIN_PASS` و `JWT_SECRET` إلى قيم خاصة بك قبل الاستخدام الفعلي، خصوصاً إذا
كان رابط الموقع متاحاً للعامة.
